# Project Partner

Live site: https://project-partner-site.netlify.app/

This repository contains the source code for the Project Partner portfolio website.

## Preview

- Live Website: https://project-partner-site.netlify.app/

## Project Structure

- `vishali-portfolio/` - main website files
- `vishali-portfolio/index.html` - homepage
- `vishali-portfolio/style.css` - styles
- `vishali-portfolio/script.js` - interactive features
- `netlify.toml` - Netlify publish configuration

## Deployment

This project is deployed on Netlify.

Publish directory:

```text
vishali-portfolio
```

## Local Run

Open `vishali-portfolio/index.html` in a browser, or run the local server file if needed.
